# 清珞未来 - AI数据服务全链路解决方案提供商

河北清珞未来信息技术有限公司（汇洲智能 002122.SZ 旗下）致力于成为 AI 模型全生命周期的数据服务商。

## 项目说明

清珞未来官方网站静态页面集合。子页面采用与首页一致的暗色浮窗导航栏风格，支持无缝跳转。

## 页面导航

| 页面 | 路径 | 主要内容 |
|------|------|---------|
| 首页 | `index.html` | 品牌展示、核心业务、行业方案、联系我们 |
| 关于我们 | `about.html` | 公司概况、战略优势、核心团队、四大基地、文化理念 |
| 数据服务 | `services.html` | 数据标注/采集、大模型数据、内容审核、服务流程 |
| 解决方案 | `solutions.html` | 智能驾驶、大语言模型、AI视频/动漫、智慧医疗 |
| 技术与保障 | `technology.html` | 自研平台、SOP流程、三级质控、项目经验 |

## 本地预览

直接用浏览器打开 `index.html` 即可。所有页面间的导航栏风格一致，点击可相互跳转。
